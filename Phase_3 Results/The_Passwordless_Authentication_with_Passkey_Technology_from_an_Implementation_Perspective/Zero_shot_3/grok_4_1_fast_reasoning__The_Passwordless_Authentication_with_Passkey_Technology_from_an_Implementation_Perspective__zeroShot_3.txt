REQUIREMENT REPORT
- Role: Backend Server
- Requirement ID: FR-01
- Requirement Title: Password Hashing with Bcrypt
- Requirement Description: Implement bcrypt hashing for passwords using salt rounds of 10 (2^10 iterations), with bcrypt.hash(password, 10) for storage and bcrypt.compare(oldPassword, user.password) for verification to produce unique, non-reversible hashes resistant to brute-force attacks.
- Citation: Page 3, Section D. Web App Security, Fig. 2
- Requirement Related Title: Password Encryption with Salting

- Role: Backend Server
- Requirement ID: FR-02
- Requirement Title: JWT Token Generation and Verification
- Requirement Description: Use jsonwebtoken library to generate JWT tokens with structure header.payload.signature, where header specifies type and algorithm, payload holds encoded user data, and signature is created via sign() method using a secret key; verify tokens on incoming requests.
- Citation: Page 3, Section D. Web App Security; Page 4, Section B
- Requirement Related Title: Token Generation

- Requirement Title: JWT Token Blacklisting
- Requirement Description: Implement MongoDB TokenBlacklist collection via Mongoose schema with fields token (String, unique) and createdAt (Date, expires: "1h"); query TokenBlacklist.findOne({ token }) to check and revoke tokens before expiration, returning "Token has been revoked" if found.
- Citation: Page 3, Section D. Web App Security
- Requirement Related Title: Token Blacklist
