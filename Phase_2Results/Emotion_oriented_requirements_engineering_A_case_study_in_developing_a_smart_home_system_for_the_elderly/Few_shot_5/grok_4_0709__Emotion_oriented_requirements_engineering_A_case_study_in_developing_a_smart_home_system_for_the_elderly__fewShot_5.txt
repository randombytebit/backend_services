Functional Requirement  
Role: System  
Requirement ID: FR-01  
Title: Learn User's Routine and Behavior  
Description: The system should learn and adapt to the user's routine and behavior using built-in AI capability, incorporating machine learning to analyze sensor data for patterns in daily activities, enabling features like anomaly detection and activity-triggered reminders.  
Citation: Section 4.1  
Related Title: Elderly Independent Living Support System  

Functional Requirement  
Role: System  
Requirement ID: FR-02  
Title: Detect Anomalous Behavior  
Description: The system should detect anomalous situations using AI/ML on sensor data, such as identifying unusual activity patterns (e.g., spending abnormal time in bed) to issue alerts and support independent living.  
Citation: Section 4.1  
Related Title: Anomaly Detection in Smart Home Environments  

Non-Functional Requirement  
Role: System  
Requirement ID: NFR-01  
Title: Accuracy of AI Component  
Description: The system should verify the accuracy of the AI component's predictions and behavior learning against ground-truth data from checklists and sensor recordings to ensure reliable anomaly detection and notifications at runtime.  
Citation: Section 4.3.1  
Related Title: AI Reliability in Passive Monitoring Systems
