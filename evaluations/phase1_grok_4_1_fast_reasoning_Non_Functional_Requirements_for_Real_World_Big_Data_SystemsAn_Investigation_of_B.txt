Title request: Real-World Big Data Systems

Non-Functional Requirement
Role: Big Data System
Requirement ID: NF-01
Requirement Title: Handle Batch Data Processing
Requirement Description: Capability to process data which is finite and usually large in volume, e.g. data archived in distributed file systems or databases, with companies processing hundreds of petabytes stored and tens of petabytes daily.
Citation: Section 3.1
Requirement Related Title: Real-World Big Data Systems

Non-Functional Requirement
Role: Big Data System
Requirement ID: NF-02
Requirement Title: Handle Stream Data Processing
Requirement Description: Capability to process data which is potentially infinite and usually flowing at high velocity, e.g. monitoring data captured and processed in real-time or close to real-time.
Citation: Section 3.2
Requirement Related Title: Real-World Big Data Systems

Non-Functional Requirement
Role: Big Data System
Requirement ID: NF-03
Requirement Title: Process Late Out-of-Order Data
Requirement Description: Capability to process data which arrives late or in a different order from that in which it was emitted, using concepts like windows of data, low watermarks, or lag methods.
Citation: Section 3.3
Requirement Related Title: Real-World Big Data Systems

Role: Big Data System
Requirement ID: NF-04
Requirement Title: Provide Processing Guarantees
