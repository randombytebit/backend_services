Title request: Real-World Big Data Systems

Non-Functional Requirement
Role: System
Requirement ID: NF-01
Requirement Title: Support Batch Data Processing
Requirement Description: The system must be capable of processing large volumes of finite data, such as data archived in distributed file systems or databases, with volumes reaching hundreds of petabytes for storage and tens of petabytes for daily processing.
Citation: Section 3.1
Requirement Related Title: Real-World Big Data Systems

Non-Functional Requirement
Role: System
Requirement ID: NF-02
Requirement Title: Enable Stream Data Processing
Requirement Description: The system must support processing of potentially infinite data flowing at high velocity, such as real-time or near real-time monitoring data, using specialized stream processing frameworks or applications.
Citation: Section 3.2
Requirement Related Title: Real-World Big Data Systems

Role: System
Requirement ID: NF-03
Requirement Title: Handle Late and Out-of-Order Data
Requirement Description: The system must be designed to process streaming data that arrives late or out of order, utilizing mechanisms like data windows or low watermarks to manage such scenarios effectively.
