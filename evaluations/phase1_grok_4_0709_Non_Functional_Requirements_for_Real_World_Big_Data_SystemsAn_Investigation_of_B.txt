Functional Requirement
Role: Big Data System
Requirement ID: FR-01
Requirement Title: Process Finite Batch Data
Requirement Description: The system must have the capability to process data which is finite and usually large in volume, such as data archived in distributed file systems or databases.

Role: Big Data System
Requirement ID: FR-02
Requirement Title: Process Infinite Stream Data
Requirement Description: The system must have the capability to process data which is potentially infinite and usually flowing at high velocity, such as monitoring data captured and processed in real-time or close to real-time.
Citation: Section 3.2
Requirement Related Title: Real-World Big Data Systems

Requirement ID: FR-03

Functional Requirement
Role: Big Data System
Requirement ID: FR-05
Requirement Title: Control Data Flow Backpressure
Requirement Description: The system must have the capability to handle scenarios where the data source is emitting records faster than the system can consume, using strategies like dropping records, sampling, or applying source backpressure.
Citation: Section 3.9
